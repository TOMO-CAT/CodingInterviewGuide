# 算法题常用位运算

## 进制表示方式

C++中，我们可以指定整数的进制：

| 进制     | 前缀 |
| -------- | ---- |
| 二进制   | 0b   |
| 八进制   | 0    |
| 十六进制 | 0x   |

## 常用位运算符

| 运算符 | 功能   | 用法           |
| ------ | ------ | -------------- |
| ~      | 位求反 | ~ expr         |
| <<     | 左移   | expr1 << expr2 |
| >>     | 右移   | expr1 >> expr2 |
| &      | 位与   | expr & expr    |
| ^      | 位异或 | expr ^ expr    |
| \|     | 位或   | expr \| expr   |

## Bitset

如果需要操作二进制数字，最好使用 Bitset 结构。
